Tentang Saya

Saya mahasiswa Sistem Informasi semester 7 yang berfokus pada pengembangan aplikasi dan pengelolaan data. Saat ini saya menjalani internship di Kementerian Agama RI pada divisi Data Sistem Informasi & Humas. Saya terlibat dalam proses validasi data siswa madrasah, perbaikan NISN, merger data, pengecekan konsistensi pada server integrasi, serta pengelolaan status aktif/non-aktif data siswa. Lingkungan kerja ini membuat saya terbiasa menangani masalah data dengan pendekatan yang sistematis, teliti, dan berbasis debugging.

Tech Stack & Tools

Untuk pengembangan web, saya menggunakan Laravel, JavaScript, MySQL, dan HTML/CSS. Saya juga memanfaatkan Tailwind, Bootstrap, dan Figma untuk membangun tampilan yang konsisten dan responsif.
